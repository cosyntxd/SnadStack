# SnadStack
This is just a small project to learn Rust, so don't expect a fully-fledged out game.
![placeholder image](placeholder.png)

# Build and Run:
```properties
git clone https://github.com/cosyntxd/SnadStack
cd SnadStack
cargo run --release
```

# Run on Web:
```properties
sh web/run.sh --release
```
